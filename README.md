<h1 align="center">Hyperion</h1>

<p align="center">A sleek, batteries included chat client for Twitch.</p>

![Preview](https://raw.githubusercontent.com/usehyperion/website/refs/heads/main/apps/web/src/lib/assets/preview.png)

> [!IMPORTANT]
> Hyperion is still under heavy development, expect bugs.

## Installing

Install the latest release from the [Releases](https://github.com/usehyperion/hyperion/releases) page.

> [!WARNING]
> Linux should run but has not been thoroughly tested.

## Building

1. Clone the repository
2. Make sure the [prerequisites](https://v2.tauri.app/start/prerequisites/) are satisfied
3. Build with `pnpm tauri build`
